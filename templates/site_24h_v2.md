# 工地24时 - 内容模板 v2

> 定位：时间的切片
> 核心问题：24小时里，有多少个瞬间值得被永久记录？
> 平台：抖音/视频号 | 频率：随项目 | 形式：短视频

---

## 标题公式

```
时间切片: 时间 + 地点 + 具体动作
示例: "凌晨4点，贵阳，浇筑开始"
```

---

## 视频脚本结构

### 时间线格式
```
[05:30] 天还没亮，工人已经在路上
[06:15] 晨会，分配任务
[07:00] 第一车混凝土到场
[10:30] 第8块墙板吊装
[12:00] 午饭，工地边上的盒饭
[14:30] 下午的浇筑
[17:45] 黄昏，收工前的验收
[18:30] 老周点了一根烟
```

### 技术要求
- 时长: 15-60秒
- 字幕: 必须有
- BGM: 安静/环境音为主
- 画面: 真实工地，不摆拍

---

## 思想厚度自检

- [ ] 有具体时间（不是"白天"，是"17:45"）
- [ ] 有真实人物（不是"工人们"，是"老周"）
- [ ] 不可替代性：只有AKO能记录这个工地
