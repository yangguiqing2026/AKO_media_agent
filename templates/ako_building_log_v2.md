# AKO建造志 - 内容模板 v2

> 定位：建造现场的人类学
> 核心问题：这个下午，工地上发生了什么只有AKO能记录的事？
> 平台：公众号/官网 | 频率：周更 | 字数：800-2000字

---

## 标题公式

```
时间切片: 时间 + 地点 + 具体动作
示例: "6:17，贵阳，吊车还没醒，但老周已经在了"
```

---

## 段落结构

### 1. 切片（1句）
> 一个具体的瞬间或数字

```
[在此写入一个具体的瞬间]
示例: "第17个模块，重4.2吨。"
```

### 2. 展开（3-5句）
> 技术/人文/时间维度

```
[在此展开技术细节]
[在此展开人物动作]
[在此展开时间/环境]

示例:
"吊车臂展35米，操作员老周说今天风大，放慢了速度。
墙板厚度120mm，面密度138kg/m²，螺栓连接。
三个人站在顶板上验收，没晃。"
```

### 3. 收束（1句）
> 回到人，或回到时间，或回到无法回答的问题

```
[在此收束，回到人或时间]
示例: "老周抽了根烟，说下个月能收工。"
或: "但谁知道下个月，风还会不会这么大？"
```

---

## 思想厚度自检

- [ ] 不可替代性：内容中"只有AKO能说的元素"占比 > 40%
- [ ] 具体性：有具体数字/具体人/具体场景
- [ ] 时间诚实：经得起时间检验
- [ ] 人的在场：有人的在场感
- [ ] 终极测试：远大住工会发这个吗？如果会，不发
