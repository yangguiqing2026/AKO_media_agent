# 数据使用政策

> 版本：v1.0
> 生效日期：____年__月__日
> 适用范围：AKO_media_agent 所有数据采集与使用

---

## 一、数据采集原则

1. **合法性**：遵守《数据安全法》《个人信息保护法》及各平台服务条款
2. **必要性**：仅采集品牌监测、趋势分析、竞品追踪所必需的数据
3. **最小化**：不采集与业务无关的用户隐私数据

## 二、数据来源分类

### 2.1 公开数据（可采集）
- 社交媒体公开帖子、评论
- 行业新闻RSS
- 各平台公开热榜/指数
- 竞品公开页面信息

### 2.2 禁止采集
- 登录后私密内容
- 用户手机号、地址等隐私
- 付费/会员内容
- 用户关系链数据

## 三、数据使用范围

| 用途 | 允许 | 说明 |
|------|------|------|
| 品牌声誉监控 | ✅ | 追踪AKO品牌提及 |
| 行业趋势分析 | ✅ | 热门话题、关键词 |
| 竞品内容分析 | ✅ | 公开内容策略分析 |
| 内容创作参考 | ✅ | 选题、标题优化 |
| 商业数据售卖 | ❌ | 严格禁止 |
| 用户画像营销 | ❌ | 严格禁止 |

## 四、数据保留政策

| 数据类型 | 保留期限 | 处理方式 |
|----------|----------|----------|
| 采集日志 | 6个月 | 到期自动删除 |
| 分析结果 | 12个月 | 归档存储 |
| 隐私数据 | 即时删除 | 采集后脱敏 |
| 敏感凭证 | 持续 | 加密存储于secrets.env |

## 五、数据安全措施

1. 传输加密：HTTPS
2. 存储加密：数据库字段级加密
3. 访问控制：基于角色的权限管理
4. 审计日志：所有数据操作可追溯

## 六、违规处理

- 发现违规采集：立即停止 + 删除数据 + 内部调查
- 发现数据泄露：24小时内通知相关方 + 启动应急预案
