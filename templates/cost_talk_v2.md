# 造价说 - 内容模板 v2

> 定位：价格的诚实
> 核心问题：357元/㎡这个数字背后，藏着什么行业不愿说的真相？
> 平台：公众号/小红书 | 频率：双周 | 字数：800-1500字

---

## 标题公式

```
数字诚实: 数字 + 身体感知 + 时间
示例: "138kg/m²，三个人，黄昏"
示例: "357元/㎡，包含什么，不包含什么"
```

---

## 段落结构

### 1. 切片（1句）
> 一个具体的价格数字

```
[在此写入一个具体的造价数字]
示例: "357元/㎡，这是桐木岭项目的实际造价。"
```

### 2. 展开
> 拆解价格构成，诚实说明包含/不包含什么

```
[价格拆解]
[行业对比]
[为什么能做到这个价格]

示例:
"包含：基础、主体、水电预埋。
不包含：装修、门窗、外墙涂料。
为什么？因为很多人报价'300元/㎡'，但那是毛坯中的毛坯。
我们选择把每一项都写清楚。"
```

### 3. 收束
> 回到诚实

```
[回到对价格的诚实态度]
示例: "价格不是越低越好，是越清楚越好。"
```

---

## 思想厚度自检

- [ ] 有具体数字（不是"性价比高"，是"357元/㎡"）
- [ ] 有行业对比（不是"我们便宜"，是"行业通常这样报价"）
- [ ] 不可替代性：只有AKO能给出这么具体的数字
