## 一、审查范围

本审查意见针对 AKO_media_agent 使用 MediaCrawler 开源工具进行社交媒体数据采集的合规性。

## 二、工具基本信息

| 项目 | 内容 |
|------|------|
| 工具名称 | MediaCrawler |
| GitHub地址 | https://github.com/NanmiCoder/MediaCrawler |
| 开源协议 | MIT License |
| Stars | 27,000+ |
| 技术原理 | Playwright 模拟真实浏览器 |

## 三、合规审查清单

- [ ] 确认MediaCrawler使用MIT开源协议，允许商业使用
- [ ] 确认使用Playwright模拟浏览器，非逆向工程
- [ ] 确认仅采集公开页面信息
- [ ] 确认不绕过任何反爬机制
- [ ] 确认不采集用户隐私信息（手机号、地址等）
- [ ] 确认不采集付费/会员内容
- [ ] 确认不批量采集用户关系链
- [ ] 确认数据仅用于学习研究和品牌监测
- [ ] 确认不售卖采集数据
- [ ] 确认遵守各平台robots.txt规定

## 四、技术合规措施

1. **请求频率控制**
   - 最小间隔：2秒/请求
   - 随机抖动：±0.5秒
   - 日总量上限：单平台≤500条

2. **代理IP使用**
   - 使用住宅代理IP
   - 轮换间隔：≥2秒
   - 地理分布：多省市分散

3. **账号管理**
   - 5个账号轮换
   - 单账号日请求≤100次
   - 登录态缓存（二维码授权）

4. **验证码处理**
   - 遇到验证码自动暂停
   - 人工介入处理

## 五、数据保留政策

- 采集日志保留：6个月
- 隐私数据：采集后立即脱敏
- 敏感凭证：存储于secrets.env，不入库

## 六、投诉响应机制

- 响应时间：24小时内
- 处理方式：立即停止相关采集 + 删除数据
- 联系人：admin@akobuild.cloud
