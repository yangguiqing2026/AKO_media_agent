# AKO人物 - 内容模板 v2

> 定位：人的在场
> 核心问题：这个人，为什么只能在AKO出现？
> 平台：公众号/LinkedIn | 频率：月更 | 字数：800-2000字

---

## 标题公式

```
人物在场: 人名 + 动作 + 一句话
示例: "老周：下个月能收工，但不知道风还大不大"
```

---

## 段落结构

### 1. 切片（1句）
> 一个人的一个动作或一句话

```
[在此写入人物的具体瞬间]
示例: "张师傅摘下安全帽，擦了擦汗，说：'这批墙板比上次好。'"
```

### 2. 展开
> 人物故事、技术细节、时间维度

```
[人物背景]
[与AKO的关系]
[具体的工作场景]
[人物的态度/感受]
```

### 3. 收束
> 回到人本身

```
[回到人物的本质]
示例: "张师傅在AKO干了三年。他说，这里的墙板让他觉得自己在做一件对的事。"
```

---

## 思想厚度自检

- [ ] 有具体人名（不是"一位工人"，是"张师傅"）
- [ ] 有具体细节（不是"工作认真"，是"摘下安全帽，擦了擦汗"）
- [ ] 不可替代性：这个人只能在AKO出现
